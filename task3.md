#### Lab1 : c program to compute sum of n numbers.

```
#include<stdio.h>

int main() {
          int i,sum=0,n=5;
          for(i=1;i<=n;++i) {
          sum= +=i;
          }
  printf("Sum of numbers from 1 to %d is %d\n",n,sum);
  return 0;

}
```
